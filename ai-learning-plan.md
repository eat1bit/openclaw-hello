# Java 程序员转行 AI - 5.5 个月学习计划

**学员情况：**
- 9 年 Java+ 大数据经验
- 每天 1h + 周末 4h
- 数学基础：高中水平
- 目标：AI 工程师

**总周期：** 160 天（约 5.5 个月）

---

## 阶段划分

### 阶段 1：Python+ML 基础（第 1-40 天）

#### 第 1-10 天：Python 速成
- 已有 Java 基础，只学语法差异
- 重点：列表推导、装饰器、numpy/pandas

#### 第 11-40 天：机器学习核心
- 吴恩达 ML 课程
- 重点：线性回归、逻辑回归、决策树、SVM、聚类

### 阶段 2：深度学习+PyTorch（第 41-70 天）

#### 第 41-55 天：神经网络基础
- 李宏毅 DL 前几讲
- 重点：反向传播、梯度下降、激活函数

#### 第 56-70 天：PyTorch 实战
- 动手学深度学习
- 重点：CNN、RNN、Transformer

### 阶段 3：项目实践（第 71-130 天）

#### 第 71-90 天：项目 1 - 大数据+AI
- Spark MLlib 用户行为预测

#### 第 91-110 天：项目 2 - 深度学习应用
- NLP 情感分析或 CV 图像分类

#### 第 111-130 天：项目 3 - Kaggle 竞赛
- Titanic 或 House Prices

### 阶段 4：求职准备（第 131-160 天）

#### 第 131-140 天：简历包装
- GitHub 整理
- 技术博客

#### 第 141-155 天：AI 面试八股
- 机器学习面试题
- 深度学习面试题
- 项目深挖

#### 第 156-160 天：模拟面试
- 刷题 + 模拟
