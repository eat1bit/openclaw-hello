# AI 学习题库

## Python 基础（第 1-10 天）

### 第 1 天：Python 基础语法
**1. Python 中定义变量时，不需要声明什么？**
- A. 变量名
- B. 数据类型 ✅
- C. 初始值
- D. 作用域

**解析：** Python 是动态类型语言，变量不需要声明类型，赋值时自动推断。

**2. 以下哪个是 Python 的正确赋值语法？**
- A. var x = 5
- B. int x = 5
- C. x = 5 ✅
- D. let x = 5

**解析：** Python 直接赋值，不需要类型声明或关键字。

**3. Python 中输出内容用什么函数？**
- A. console.log()
- B. print() ✅
- C. echo
- D. System.out.println()

**解析：** Python 用 print() 函数输出，console.log 是 JavaScript 的。

---

### 第 2 天：Python 控制流
**1. Python 中条件判断用什么关键字？**
- A. when
- B. if ✅
- C. case
- D. switch

**解析：** Python 用 if/elif/else 进行条件判断，没有 switch-case。

**2. 以下哪个是正确的 if 语句语法？**
- A. if (x > 5)
- B. if x > 5 then
- C. if x > 5: ✅
- D. if x > 5 {}

**解析：** Python 的 if 语句用冒号结尾，不用括号和花括号。

**3. Python 中等号比较用？**
- A. =
- B. == ✅
- C. ===
- D. equals

**解析：** = 是赋值，== 是比较，Python 没有===。

---

### 第 3 天：Python 函数
**1. Python 中定义函数用什么关键字？**
- A. function
- B. def ✅
- C. func
- D. define

**解析：** Python 用 def 关键字定义函数。

**2. 函数参数默认值在哪里指定？**
- A. 函数定义时 ✅
- B. 函数体内
- C. 返回值后
- D. 不能指定

**解析：** Python 可以在定义函数时指定参数默认值。

**3. 导入模块用什么关键字？**
- A. include
- B. require
- C. import ✅
- D. using

**解析：** Python 用 import 导入模块，include 是 C 语言的。

## 机器学习（第 11-40 天）

### 基础概念
**1. 线性回归用于什么类型的问题？**
- A. 分类
- B. 回归 ✅
- C. 聚类
- D. 降维

**解析：** 线性回归用于预测连续值的回归问题。

**2. 过拟合是指模型在什么上表现好？**
- A. 训练集 ✅
- B. 测试集
- C. 验证集
- D. 新数据

**解析：** 过拟合是模型在训练集上表现太好，泛化能力差。

**3. 哪个不是机器学习算法？**
- A. 决策树
- B. SVM
- C. Python ✅
- D. KMeans

**解析：** Python 是编程语言，不是算法。

**4. 训练集和测试集的常见划分比例是？**
- A. 50% : 50%
- B. 80% : 20% ✅
- C. 10% : 90%
- D. 99% : 1%

**解析：** 通常 80% 数据用于训练，20% 用于测试，保证模型有足够数据学习且有独立数据验证。

**5. 特征工程的主要目的是？**
- A. 增加数据量
- B. 提高模型性能 ✅
- C. 减少计算时间
- D. 简化代码

**解析：** 特征工程通过选择、转换特征来提高模型预测性能。

### 数据处理
**6. pandas 中读取 CSV 文件用什么函数？**
- A. read_excel()
- B. read_csv() ✅
- C. load_csv()
- D. import_csv()

**解析：** pandas 用 pd.read_csv() 读取 CSV 文件。

**7. numpy 数组和 Python 列表的主要区别是？**
- A. 数组只能存数字 ✅
- B. 数组更高效 ✅
- C. 列表不能切片
- D. 没有区别

**解析：** numpy 数组针对数值计算优化，支持向量化操作，效率更高。（注：A 和 B 都对，但 B 是核心优势）

**8. 数据标准化的主要作用是？**
- A. 让数据范围统一 ✅
- B. 增加数据量
- C. 删除异常值
- D. 转换数据类型

**解析：** 标准化将不同量纲的特征转换到同一尺度，避免某些特征主导模型。

### 模型评估
**9. 分类问题最常用的评估指标是？**
- A. 均方误差
- B. 准确率 ✅
- C. R²分数
- D. 平均绝对误差

**解析：** 准确率是分类问题最直观的评估指标，表示预测正确的比例。

**10. 混淆矩阵中 TP 表示什么？**
- A. 真阳性 ✅
- B. 假阳性
- C. 真阴性
- D. 假阴性

**解析：** TP (True Positive) 表示实际为正例且预测也为正例的数量。

**10. 防止过拟合的常用方法是？**
- A. 增加训练轮数
- B. 使用正则化 ✅
- C. 减少训练数据
- D. 增加模型复杂度

**解析：** 正则化 (L1/L2) 通过惩罚大权重来防止过拟合， Dropout 也是常用方法。

## Python 实战

**1. 以下代码的输出是什么？`print([i for i in range(3)])`**
- A. [1, 2, 3]
- B. [0, 1, 2] ✅
- C. [1, 2]
- D. [0, 1, 2, 3]

**解析：** range(3) 生成 0,1,2，列表推导式创建列表 [0, 1, 2]。

**2. Python 中处理异常用什么语句？**
- A. try-catch
- B. try-except ✅
- C. do-catch
- D. if-error

**解析：** Python 用 try-except 处理异常，不是 try-catch（那是 Java/JavaScript）。
