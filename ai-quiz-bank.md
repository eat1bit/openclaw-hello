# AI 学习题库

## Python 基础（第 1-10 天）

### 第 1 天：Python 基础语法
**1. Python 中定义变量时，不需要声明什么？**
- A. 变量名
- B. 数据类型 ✅
- C. 初始值
- D. 作用域

**解析：** Python 是动态类型语言，变量不需要声明类型，赋值时自动推断。

**2. 以下哪个是 Python 的正确赋值语法？**
- A. var x = 5
- B. int x = 5
- C. x = 5 ✅
- D. let x = 5

**解析：** Python 直接赋值，不需要类型声明或关键字。

**3. Python 中输出内容用什么函数？**
- A. console.log()
- B. print() ✅
- C. echo
- D. System.out.println()

**解析：** Python 用 print() 函数输出，console.log 是 JavaScript 的。

---

### 第 2 天：Python 控制流
**1. Python 中条件判断用什么关键字？**
- A. when
- B. if ✅
- C. case
- D. switch

**解析：** Python 用 if/elif/else 进行条件判断，没有 switch-case。

**2. 以下哪个是正确的 if 语句语法？**
- A. if (x > 5)
- B. if x > 5 then
- C. if x > 5: ✅
- D. if x > 5 {}

**解析：** Python 的 if 语句用冒号结尾，不用括号和花括号。

**3. Python 中等号比较用？**
- A. =
- B. == ✅
- C. ===
- D. equals

**解析：** = 是赋值，== 是比较，Python 没有===。

---

### 第 3 天：Python 函数
**1. Python 中定义函数用什么关键字？**
- A. function
- B. def ✅
- C. func
- D. define

**解析：** Python 用 def 关键字定义函数。

**2. 函数参数默认值在哪里指定？**
- A. 函数定义时 ✅
- B. 函数体内
- C. 返回值后
- D. 不能指定

**解析：** Python 可以在定义函数时指定参数默认值。

**3. 导入模块用什么关键字？**
- A. include
- B. require
- C. import ✅
- D. using

**解析：** Python 用 import 导入模块，include 是 C 语言的。

## 机器学习（第 11-40 天）

### 基础概念
**1. 线性回归用于什么类型的问题？**
- A. 分类
- B. 回归 ✅
- C. 聚类
- D. 降维

**解析：** 线性回归用于预测连续值的回归问题。

**2. 过拟合是指模型在什么上表现好？**
- A. 训练集 ✅
- B. 测试集
- C. 验证集
- D. 新数据

**解析：** 过拟合是模型在训练集上表现太好，泛化能力差。

**3. 哪个不是机器学习算法？**
- A. 决策树
- B. SVM
- C. Python ✅
- D. KMeans

**解析：** Python 是编程语言，不是算法。
